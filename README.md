# Restaurang Felicé

A website for a restaurant in Växjö, Sweden. An example of how uJet and uJetMaps can be used.

## Getting started

1. Get the source code.
2. Unzip `UmbracoCms.7.0.4.zip` (do not overwrite the `Web.config` file).
3. Compile the source code and run the website.
